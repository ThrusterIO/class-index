The Thruster ClassIndex Component.


## Install

Via Composer

```bash
$ composer require thruster/class-index
```

## Testing

Run test cases

```bash
$ composer test
```

Run test cases with coverage (HTML format)


```bash
$ composer test-coverage
```

Run PHP style checker

```bash
$ composer check-style
```

Run PHP style fixer

```bash
$ composer fix-style
```
